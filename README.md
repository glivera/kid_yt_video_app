# Детские YouTube Видео

React приложение для безопасного просмотра YouTube видео детьми. Родители могут выбирать и утверждать видео, а дети смотреть только одобренный контент.

## Описание проекта

Приложение предоставляет два типа аккаунтов:

### Родительский аккаунт
- Поиск видео по ключевым словам через YouTube API
- Рекомендации по темам для развития ребенка (обучение, творчество, наука и т.д.)
- Утверждение подходящих видео для ребенка
- Блокировка нежелательных видео и каналов
- Управление списками доступного контента

### Детский аккаунт
- Просмотр только утвержденных родителями видео
- Безопасный видеоплеер
- История просмотров
- Простой и понятный интерфейс

## Технологический стек

- **React** 18.2 - библиотека для создания пользовательского интерфейса
- **React Router** 6.20 - маршрутизация
- **Vite** 5.0 - сборщик и dev-сервер
- **Axios** - HTTP клиент для работы с YouTube API
- **YouTube Data API v3** - поиск и получение информации о видео

## Установка и запуск

### Предварительные требования

- Node.js (версия 16 или выше)
- npm или yarn
- YouTube Data API ключ

### Получение YouTube API ключа

1. Перейдите на [Google Cloud Console](https://console.cloud.google.com/)
2. Создайте новый проект или выберите существующий
3. Включите YouTube Data API v3
4. Создайте credentials (API ключ)
5. Скопируйте ключ

### Установка зависимостей

```bash
npm install
```

### Настройка окружения

1. Создайте файл `.env` в корне проекта:
```bash
cp .env.example .env
```

2. Добавьте ваш YouTube API ключ:
```
VITE_YOUTUBE_API_KEY=your_api_key_here
```

### Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

### Сборка для продакшена

```bash
npm run build
```

Готовые файлы будут в папке `dist/`

### Предварительный просмотр продакшен сборки

```bash
npm run preview
```

## Архитектура приложения

### Маршрутизация

- `/login` - страница входа
- `/parent/*` - родительская панель (защищенный роут)
  - `/parent/search` - поиск видео
  - `/parent/approved` - утвержденные видео
  - `/parent/blocked` - заблокированный контент
  - `/parent/recommendations` - рекомендации по темам
- `/child/*` - детская панель (защищенный роут)
  - `/child/videos` - список доступных видео
  - `/child/player/:videoId` - просмотр видео
  - `/child/history` - история просмотров

### Управление состоянием

- **AuthContext** - управление авторизацией и ролями пользователей
- **localStorage** - хранение данных:
  - Информация о пользователе
  - Утвержденные видео
  - Заблокированные видео и каналы
  - История просмотров

### Безопасность

- Защищенные роуты с проверкой роли пользователя
- SafeSearch включен в YouTube API запросах
- Фильтрация видео по возможности встраивания
- Родительский контроль над доступным контентом
